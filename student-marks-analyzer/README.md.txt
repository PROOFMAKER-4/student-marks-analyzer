# Student Marks Analyzer

A Python project to analyze student marks, compute quartiles, and visualize results using matplotlib.

## Features
- Reads marks from CSV files
- Computes mean, median, quartiles
- Generates boxplots
- Handles invalid input gracefully

## Installation
```bash
git clone https://github.com/yourusername/student-marks-analyzer.git
cd student-marks-analyzer
pip install -r requirements.txt
